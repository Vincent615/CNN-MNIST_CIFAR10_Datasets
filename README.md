# CNN_MNIST-CIFAR10
**This is generally a LeNet** with accuracy of ~98.5% on MNIST dataset and ~70% on CIFAR10 dataset.

train.py is the code to train and test the model. 

## Prerequisites
- Pytorch
- Torchvision
- Cuda
